Estudando Html


Nossos estudos começaram falando a respeito de um item muito sério que é a organização. Isso porque a linguagem HTML é responsável


Paragráfos

Para colocarmos parágrafos, usamos os elementos "p" e "/p". Para fazer uma quebra de linha, mantendo o mesmo parágrafo, usamos o elemento "br". Temos ainda parágrafos especiais, denomonados títulos, cabeçalhos ou headlings em que são usados elementos "h1" a "h6" e seus respectivos fechamentos de "/h1" a "/h6"


Imagens

Uma imagem aparece na janela do navegador pelo uso do elemento "img". Esse elemento possui o atribudo "src", responsavel pela localização da imagem. Para trabalhar melhor com a imagem, você pode coloca-la dentro das tagas de marcação da página
